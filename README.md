# 情绪自动反应打破器（手机小工具）

这是一个离线可用、数据只在本机保存的小 PWA，用来把你在被触发时的自动反应拆开：

**触发 → 解读 → 情绪 → 行为 → 后果**

核心目标：在“解读”这一段插入替代解读，配合 **4 步流程（识别 / 暂停 / 重写解读 / 自我安抚）**，把循环切开。

现在包含一条清晰的用户路径（底部导航）：

- **当下**：先写“发生了什么”，按底层逻辑拆解（触发/解读/情绪/行为/最坏画面）→ 生成“行动卡”→ 保存为一次打卡
- **舒缓**：两种方式稳定情绪：书写情绪（写完给解读）/ 白噪音（闭眼放松）
- **打卡**：按周与按月汇总，帮助你看到事件类型与情绪强度的关联

## 本地运行

这是纯静态站点，不需要安装依赖。任选其一：

- `python3 -m http.server 5173`
- `npx serve .`

然后用手机和电脑同一局域网访问（或直接在电脑浏览器打开）。

## 安装到手机主屏幕

- iOS Safari：分享 → 添加到主屏幕
- Android Chrome：菜单 → 安装应用

## GitHub Pages 发布

这个仓库已经准备好了 GitHub Pages 的发布目录：

- 发布文件：`docs/index.html`
- 禁用 Jekyll：`docs/.nojekyll`

推荐在 GitHub 仓库中将 Pages 来源设置为：

- Branch: `main`
- Folder: `/docs`

发布后，Brera 指南会作为站点首页打开。

## 隐私

- 不上传任何数据
- 记录保存在浏览器 `localStorage` 中
